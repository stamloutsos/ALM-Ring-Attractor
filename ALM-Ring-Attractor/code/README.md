# Analysis Code

## Usage
1. Download DANDI 000060 to `../data/raw/`
2. Run: `pip install -r requirements.txt`
3. Run: `python finaloutput2DRing.py`
4. Outputs saved to `../paper/figures/`

## Main script
`finaloutput2DRing.py` : Loads NWB files, runs PCA, computes ring statistics and T_theta for hit/error trials.
